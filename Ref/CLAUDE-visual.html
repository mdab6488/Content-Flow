9 Analytics & Scoring Engine

File at /lib/scoring.ts — calculates all 4 scores:

affiliateScore (0–100)
  • cpcScore: >$4 → 40pts, >$3 → 35, >$2 → 28, >$1.50 → 20, else 10
  • densityScore: >10 links → 30, 7–10 → 22, 4–6 → 15, 1–3 → 8
  • networkScore: Amazon=20, ShareASale=18, Direct=15, None=0
  • Capped at 100
youtubeScore (0–100)
  • volumeScore: >5K→35, >2K→28, >500→20, else 10
  • competitionScore: <40→40, <60→28, <80→15, else 5
  • freshnessScore: >18mo→25, >12mo→18, >6mo→10, else 3
trendScore (0–100)
  • growthScore: >200%→50, >100%→40, >50%→28, >0%→15, negative→0
  • directionScore: EXPLODING→30, RISING→22, STABLE→12, FALLING→0
  • stabilityScore: >6mo→20, else proportional
overallScore
(affiliate × 0.35) + (youtube × 0.30) + (trend × 0.25) + (volumeNormalized × 0.10)
estRevenuePerMonth
(monthlySearches × 0.03 × conversionRate × avgCommission)
conversionRate: CPC >$3 → 4%, >$1.5 → 3%, else 2%
Returns integer dollars/month.
Export: calculateAllScores(keywordData)
→ { affiliateScore, youtubeScore, trendScore, overallScore, estRevenuePerMonth }

10 Freshness Score Calculator

File at /lib/freshness.ts — 0–100 score for competitor articles.

Recency (40pts)
  • <30 days → 40
  • <90 days → 32
  • <180 days → 22
  • <365 days → 12
  • >365 → 4 | None → 0
Word Count (25pts)
  • >3K → 25, >2K → 20
  • >1.5K → 15, >1K → 10
  • <1K → 5
Product Count (20pts)
  • >8 → 20, 5–8 → 15
  • 3–4 → 10, 1–2 → 6, 0 → 0
Publish Date (15pts)
  • <6mo → 15, <12mo → 12
  • <24mo → 8, >24mo → 3
Result: ≥70 → "Fresh" (green), ≥40 → "Aging" (yellow), <40 → "Stale" (red)
